レッスンのポイント

There is / There are　は「存在する」ことを表し、「▼▼に　〇〇　があります」また、それらが「いくつあるか、どれくらいあるか」を説明するときに使用します。

また、このレッスンでは名詞の ・数えられるモノ countable nouns ・数えられないモノ uncountable nouns の２種類についても学んでいきます。

Countable nounsの場合：There is /areは「名詞がいくつあるか」によって使い分け

There is a 〇〇. (〇〇がひとつある

There are many 〇〇s. (〇〇がたくさんある)

There are many people. (たくさんの人がいる)

その他の例　cars, projects, meetings, men, women, children, feet, teeth

Uncountable nounsの場合：There is　を使用

There is some water.

There is a lot of money.

その他の例　time, coffee, tea, space, noise, knowledge, rice, love, advice, furniture

SEE
Use “there is” or “there are” to say that something exists.

[there is/ある」または「there are/ある」を使用して、何かが存在することを伝えます

Countable nouns

数えることができる名詞

There is a problem.

問題があります。

There’re (only) a few cars.

車は（数台しか）ありません。

There are many employees.

社員の方がたくさんいます。

There aren’t any cafes.

カフェがありません。

Questions

疑問形

Is there a printer?

プリンターはありますか？

Are there any questions?

何か質問はありますか？

Uncountable nouns

数えられない名詞

There is some noise.

多少のノイズがあります。

There’s (only) a little space.

（少しだけ）スペースがあります。

There is a lot of traffic.

交通量が多いです。

There isn’t any time.

時間が全くありません

Questions

疑問形

Is there any tea?

お茶はありますか？

Example 例

Ken:

This is our department. There are 17 employees and a few temporary workers. Now, there are a few problems. There are many people, but there are only two meeting rooms. 

Also, there is a projector in Room A, but there isn’t a projector in Room B.

ケン：

私たちの部署です。社員は17名、派遣社員は数名です。今、いくつかの問題があります。人数は多いのですが、会議室が2つしかありません。また、A室にはプロジェクターがありますが、B室にはプロジェクターがありません。

Bill:

I see. By the way, is there any coffee?

ビル:

そうなんですね。ところで、コーヒーはありますか。

Ken:

Yes, there is some coffee in the kitchen. It’s this way.

ケン：

はい、キッチンにコーヒーがあります。こちらです。
