一般動詞の過去形【規則変化】

(I WORKED, I LIVED, I STARTED)

3

日本語サポートページ

レッスン理解を深めるための予習用教材です。オンラインレッスンでは1ページの英語のみの教材を使用します。

レッスンのポイント

過去の行動を表現する「〇〇しました」という動詞の過去形には4つの種類があります。

【v. +ed】

　 ① Worked (発音：t)　asked, danced, increased, pushed, reached, fixed, laughed など

　 ② Lived(発音：d)　entered, received, raised, traveled, explained, cried など

　 ③ Started(発音：id)　connected, started, decided, endedなど

【その他】

　 ④ ate, spoke, went　など ※レッスン4で練習します。

SEE
You can talk about the past by adding “-ed” to some verbs.
It’s pronounced in three different ways.

一部の動詞に「-ed」をつけることで過去の話をすることができます。発音は3通りあります。

/t/

　I talked with Tom this morning. / My boss checked my report yesterday.

　

　　私は今朝トムと話した。/私の上司は昨日私のレポートをチェックしました。

/d/

　I lived in Hokkaido five years ago. / We used fax machines a long time ago.

　

　　5年前に北海道に住んでいました。/昔はFAXを使っていました。

/id/

　We started a new project last summer. / The meeting ended at 1:00.

　

　　去年の夏から新しいプロジェクトを始めました。/ 会議は1時に終了しました。

You use “did” for questions and negative answers.

質問や否定的な答えには「did」を使います。

A:　

Did you enjoy the movie?

　

　 映画は楽しめましたか？

B:　

No, I didn’t enjoy it.

　

　 いえ、楽しめませんでした。

Example例

Bill:

How did the meeting go?

ビル:

会議はどうでしたか。

Ken:

It went well. We discussed our new project. We needed more people, so we decided to hire a part-timer.

ケン：

うまくいきました。新しいプロジェクトの話をしました。人手が足りないので、バイトを雇うことにしました。

Bill:

That’s a good idea. Did you talk about the project schedule?

ビル：

それはいい考えですね。プロジェクトのスケジュールの話はしましたか。

Ken:

We tried to talk about it, but we didn’t have time. So we scheduled another meeting.

ケン：

話し合おうとしたんですが、時間がなかったんです。それで、別のミーティングを予定していました。
