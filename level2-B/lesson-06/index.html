Level 2 - Rank B - Lesson 6: Offering assistance

Biz Tip

When showing someone how to do something, instead of using commands, like “Push the start button.”, say “We push the start button.”

Biz Tip

Try to avoid saying “No” or “You can’t do that.” Instead, you can say “This machine doesn’t have that function.”

See

Example

Taro:

Hi Li. Do you need some help using the photocopier?

Li:

Yes, thanks. All the instructions are in Japanese.

Taro:

OK, put your document here.

Li:

OK. And I just push the start button right?

Taro:

No, no, no. What size of copy do you want?

Li:

Oh, OK. A4 size, please. I have two documents, but I want them to be on the same sheet. Can I do that?

Taro:

No, you can’t.

Li:

I see.

Taro:

Then, push the start button.

Li:

That’s easy. Thanks for your help.

Bizmates Style

Taro:

Hi Li. Do you need some help using the photocopier?

Li:

Yes, thanks. All the instructions are in Japanese.

Taro:

OK, first we place your document here.

Li:

OK. And I just push the start button right?

Taro:

Before that, we have to decide the copy size.

Li:

Oh, OK. A4 size please. I have two documents, but I want them to be on the same sheet. Can you do that?

Taro:

Sorry, I don’t think this copier has that function.

Li:

I see.

Taro:

Then, we push the start button.

Li:

That’s easy. Thanks for your help.
