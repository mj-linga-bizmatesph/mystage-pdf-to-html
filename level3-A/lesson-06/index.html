Level 3 - Rank A - Lesson 6: Applying project management tools

Biz Tip

A useful tool to manage:
TIME: use “3-point estimating” to prepare for uncertainty.
MONEY: do reforecasting as the project progresses.
PEOPLE: hold daily scrum meetings to improve communication.

Biz Tip

The project manager needs to be constantly monitoring and making adjustments to the schedule and budget and communicating these to his team.

See

Example

Yasu:

OK, now I understand that as project manager, I need to carefully manage the three resources of time, money, and people. But do you have any practical tips?

Frank:

OK, let’s start with time. You can use “3-point estimating” when creating your project schedule.

Yasu:

What do you mean?

Frank:

It just means to create 3 schedules, a worst-case scenario with delays, a best-case with no delays, and a realistic one.

Yasu:

That makes sense. What about money?

Frank:

Well, you need to closely manage your budget. And remember that forecasting is an ongoing process. Continuously reforecast and communicate this to your team.

Yasu:

Right. The problem is the executives keep making new requests.

Frank:

Well, you need to reforecast the budget every time the project scope changes. If your project scope changes, the budget and schedule need to be adjusted too.

Yasu:

I see. What about communication with my project team?

Frank:

One thing that has worked for me is to have daily “scrum meetings”.

Yasu:

A meeting every day?

Frank:

Yes, just 15 minutes to answer 3 questions. What did you do yesterday? What will you do today? And are there any issues? You can do this online or face-to-face.

Yasu:

I think I get it now. As project manager, I need to be constantly monitoring and making adjustments to the schedule and budget and communicating these to my team.

Try

Answer these questions.

  • What are some possible issues that can affect project schedules?
  • What are some possible issues that can affect project budget?
  • What are some possible issues that can affect communication among a project team?
