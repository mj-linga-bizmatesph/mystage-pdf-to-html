LESSON 13
13
I HAVE A REQUEST リクエストする
In this lesson, you will learn phrases to make a request in the lesson. このレッスンでは、レッスン中に要望を伝えるためにつかうフレーズを学びます。

オンラインレッスンを受講する前に、以下の会話例を読みましょう。
オンラインレッスンでは、トレーナーと実際に会話例を練習します。
SEE
1

Making requests: Reviewing

復習をリクエストする

Are you ready for the lesson?

レッスンを始める準備はいいですか？

Can we start the lesson from the top?
I want to review.

このレッスンを頭から始められますか？
前回の復習がしたいです。

Other phrases
こんな言い方も！

Is it okay to start again? I want to review.

また初めからいいですか？前回の復習がした いです。

Of course. Let’s start.

もちろんです。では始めましょう。

2

Making requests: Moving on

先に進むようリクエストする

Do you have any questions about the SEE part?

SEE パートについて何か質問は ありますか？

I’m okay. Can we move on to the next part?

大丈夫です。次のパートに進めますか？

3

Making requests: Asking for an example

例を出すようリクエストする

Do you understand “worry”?

“Worry” の意味はわかりますか？

Can you give me some example sentences?

例をいくつかいただけますか？

Other phrases
こんな言い方も！

I’d like to see some example sentences.

例文をいくつか見たいです。

Of course. Let me write some in the chat box.

もちろんです。チャットボックスに いくつか書いていきますね。

Thank you. I appreciate that.

ありがとうございます。感謝します。

Other phrases
こんな言い方も！

Thanks. That’s very kind of you.

ありがとう。助かります。

TRY

Your trainer will say the following. Respond.

トレーナーが次の文章を言います。適切に返しましょう。

  1. Are you ready for the lesson? (want to make a request)
  2. Sure. How can I help you? (want to start from the top)
  3. Do you have any questions about the SEE part? (want to move on)
  4. Do you understand “worry”?
  5. Let me type some examples in the chat box.
