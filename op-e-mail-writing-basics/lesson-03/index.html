USING AN APPROPRIATE GREETING
BIZ TIP

Set the tone of your e-mail message right from the beginning by using the key phrases below:

Formal:
  • Dear Mr. Jones:
  • Dear Sir,
  • Hi, Ms. Atkins.
Casual:
  • Hey, Jim.
  • Hi, Bob.
  • Tommy,
