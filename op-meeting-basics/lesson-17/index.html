17
SUMMARIZING KEY POINTS
BIZ TIP

Make sure all members understand today’s meeting by recapping the key points and decisions. Use the expressions below:

  • Let’s summarize today’s meeting. We discussed…
  • We agreed to…
  • Does anyone have any questions before we wrap up?
SEE

As the meeting comes to a close, Ken recaps today’s main points…

Ken:
Okay, let’s summarize today’s meeting. We discussed the project concept and we agreed to create a product that will link all of our existing home products together. In addition, we agreed to build a new system that will allow us to gather data from these devices. Does anyone have any questions before we wrap up?
All:
[No questions]
Ken:
Okay, regarding the next meeting, …
