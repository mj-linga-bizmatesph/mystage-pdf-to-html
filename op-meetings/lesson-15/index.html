PROJECT MEETINGS
15
BIZ TIP
The goal of project meetings is to keep track of a certain project until it has been completed. It is essential to include the following information:
  • Completed tasks
  • Tasks still in progress
  • Challenges that might delay or stop the project
  • Action plans
SEE

Example:

  • Reporting completed tasks

    I’m pleased to report that we’ve already completed…

    We’ve finally finished…

  • Reporting work in progress

    The team is now in the process of putting together…

    We’ve already begun working on…

  • Reporting challenges

    Currently, the project is faced with…

    I’m afraid we are running behind schedule due to…

  • Reporting action plans

    As a work-around, we’ve developed…

    To avoid any further delays, we’ve set up a system that…

TRY

Answer these questions.

  1. What information does your boss ask for during project meetings?
  2. How do you feel when a project falls behind schedule?
  3. Which of the 4 key information above would you likely focus on if you were the project manager? Why?
