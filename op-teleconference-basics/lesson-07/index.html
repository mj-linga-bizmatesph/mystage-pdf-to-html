7
WELCOMING PARTICIPANTS
BIZ TIP

Greet your participants and create a professional yet friendly atmosphere by using the following expressions:

  • Hello, everyone. Thank you for joining today’s teleconference.
  • Can everyone please check in to make sure we’re all here?
  • Does everyone have a copy of the agenda?
SEE
