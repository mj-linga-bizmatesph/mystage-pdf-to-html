14
GIVING INSTRUCTIONS
BIZ TIP

Making sure everyone is looking at the same file or correct documents in a teleconference can be challenging. Use the following expressions to make your directions clear and easy to understand:

  • Did everyone receive the… (that) I sent?
  • Could I have everyone…?
  • I’d like to…
SEE

Ken and the teleconference participants are sharing information and reports…

Ken
Did everyone receive the Excel file titled, “Consolidated Shipping Report” that I sent?
All
[Yes]
Ken
Great. Could I have everyone open the file and refer to the first table, please?
Rob
Sorry, Ken. I can’t seem to open the file. The password isn’t working.
Ken
Okay, let me send you the file and password again. Just a moment, please.
[ a few moments later ]
How about now, Rob?
Rob
Okay. I got it. Thanks.
Ken
Perfect. I’d like to go through the numbers and dates with…
