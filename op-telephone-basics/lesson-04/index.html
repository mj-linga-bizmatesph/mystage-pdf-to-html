OTHER PROGRAMS TELEPHONE BASICSILESSON 4
ENDING THE PHONE CALL
4
BIZ TIP
Maintain professionalism throughout the call. End the phone call by using the expressions below:
Is there anything else I can help you with?
You're welcome. Glad I could help.
Have a great day. Goodbye.
SEE
Clara
I’m really happy this has been sorted out.
Miki
Is there anything else I can help you with, Clara?
Clara
I think that’s it, Miki. Thank you so much.
Miki
You’re welcome. Glad I could help. Please call us back if it happens again.
Clara
Will do. Thanks again.
Miki
Have a great day. Goodbye, Clara.
Clara
Bye, Miki.
